# Виртуальный тур по комнате электрооборудования университета в дизайне фирмы «Россети» с интерактивным сценарием использования электронно-измерительного устройства
Программное решение является desktop приложением для визуализации брендированный аудитории компании "Россети". Рендеры и интерактивная прогулка необходима для демонстрации проекта и плана ремонта заказчику - руководство университета и фирма партнер. Дополнительно в проекте реализованы интерактивные механики измерения электрических характеристик отдельных устройств с использованием мультиметра.

 ## <b>Визуализация помещения</b>
  <p>Аудитория состоит из 1 комнаты. На территории расположены выставочные элементы электронной техники, мебель (столы, стулья и тумбы), а также брендированная символика</p>
